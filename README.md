Різниця у видачі під час використання INNER та LEFT / RIGHT JOIN

1. Коли всі таблиці були об'єднані за допомогою INNER JOIN, ми отримали 518 рядків. Результат включав лише рядки, які мають спільні атрибути серед всіх таблиць. 

2. Змінивши запит на комбінацію INNER / LEFT / RIGHT JOIN, або замінивши всі INNER JOIN на LEFT / RIGHT кількість рядків залишилась 518. 

Висновок: кількість рядків не змінилася, бо зв'язки даних у всіх таблицях повні, немає зайвих даних (постачальників чи продуктів, що ніколи не потрапляли до лянцюжка продажів), а також немає пропусків NULL, відповидно INNER нічого не зрізає.
